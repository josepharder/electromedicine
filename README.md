# Electromedicine

Jupyter notebooks for biomedical instrumentation analysis.

## Notebooks

- **biostatistics.ipynb** - Calculates mean, variance, standard deviation, Pearson correlation, and calibration curve (y = mx + b) from input/output data pairs.

- **second_order_instrument.ipynb** - Analyzes the frequency response of a second-order instrument, computing magnitude and phase shift for DC, cosine, and sine input signals.
